Come funziona?

  • • I modelli di linguaggio (LLM) generano testo un token alla volta
  • • Un token può essere una parola, parte di parola o punteggiatura
  • • Il modello predice quale token è più probabile in base al contesto precedente
  • • Ogni token ha una probabilità associata: il modello può scegliere percorsi diversi
  • • Verde = percorso principale, Arancione = percorso alternativo, Giallo = token corrente
